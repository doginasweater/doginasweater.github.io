Dog In A Sweater Tutorials

This is a tutorial to take you from zero to competent using my chosen set of development tools. Generally speaking, I'm a flexible .net dev who does way too much front-end, so this will focus a lot on my favourite tools. The main focuses of this tutorial will be:

  1. Bootstrap 4 - the utility classes it provides and the sass configuration it allows have REALLY sold me on it again.
  2. Typescript - types are amazing and I won't hear you say otherwise. Also, I think tsc is way easier to configure than babel.
  3. React - react is COOL and also component architectures are nice.
  4. ASP.NET Core - C# is great, asp.net core is great, cross-platform is great.
